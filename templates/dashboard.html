Rendimiento del Sistema
Características Implementadas
  • Endpoints API separados para acumulación y procesamiento
  • Sistema asíncrono sin activaciones automáticas
  • Extracción posicional inteligente con coordenadas
  • Validación automática de campos obligatorios
  • Procesamiento por lotes con metadatos WhatsApp
  • Perfiles de rendimiento optimizados (ultra_rapido, rapido, normal)
  • Modelos ONNX-TR para máxima eficiencia en CPU
  • Monitoreo de recursos en tiempo real
  • Integración completa con n8n workflows
  • Sistema de validación de recibos de pago
Endpoints API Principales
POST /api/ocr/process_image

Acumula imágenes individuales en el sistema para procesamiento posterior

curl -X POST http://localhost:5000/api/ocr/process_image \ -F "image=@recibo.jpg" \ -F "caption=Pago realizado" \ -F "sender_id=user123" \ -F "sender_name=Juan Pérez"
POST /api/ocr/process_batch

Procesa lote acumulado bajo demanda con control total por n8n

curl -X POST http://localhost:5000/api/ocr/process_batch \ -H "Content-Type: application/json" \ -d '{"batch_size": 5, "profile": "ultra_rapido"}'
GET /api/ocr/result/{request_id}

Consulta resultado de procesamiento individual

curl -X GET http://localhost:5000/api/ocr/result/sender123_juan_20250706_001_1430_20250706_143022_456
GET /api/ocr/queue/status

Estado completo del sistema de colas

curl -X GET http://localhost:5000/api/ocr/queue/status

Configuración para n8n:
PASO 1: Guardar Imagen

Sube tu recibo de WhatsApp. Se guarda en cola esperando procesamiento. No se procesa automáticamente.

POST /api/ocr/process_image → Devuelve: request_id único
PASO 2: Activar Procesamiento

TÚ DECIDES CUÁNDO: Procesa todas las imágenes acumuladas cuando quieras.

POST /api/ocr/process_batch → Control total de timing
PASO 3: Obtener Resultados

Consulta resultados: monto, fecha, referencia y datos del recibo extraídos automáticamente.

GET /api/ocr/result/request_id → Texto completo + datos estructurados
¿Por Qué Este Sistema?
  • Control total: Tú decides cuándo procesar, sin automatización forzada
  • Para WhatsApp: Diseñado específicamente para recibos de pago móviles
  • Sin pérdidas: Las imágenes se guardan hasta que las proceses
  • Extracción inteligente: Detecta montos, fechas, referencias automáticamente
  • Integración n8n: Perfecto para workflows automatizados
